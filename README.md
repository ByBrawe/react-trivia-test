# react-trivia-test

Trivia test application

[Demo](https://react-trivial-test-b89d8dccc772.herokuapp.com/)



# Getting started

- Clone this repo
- `npm install` to install all required dependencies
- `npm run start`

## Dependencies
- Axios
- Tailwindcss
- Tailwind-scrollbar
- react-router-dom

# Application Features
- Trivia Test
- Difficulty Mod
- Skor calculation
- Result and Your marked questions and answers list
